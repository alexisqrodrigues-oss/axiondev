# Welcome to axison.dev

This is the first post. The idea is simple: **document everything** I learn in dev and AI.

## Why this blog exists

Writing is the best way to **lock in** knowledge. I solve problems, forget the solution, and three months later go through the same pain again. This blog is my antidote.

> "_If you didn't write it, you don't know it._"

## What you'll find here

- Workflows and productivity tips
- Software setup
- Academic study notes
- Scripts (bash, python, node)
- AI automations
- Step-by-step tutorials
- LLM benchmarks

## How it works under the hood

Pure **markdown**. Each post lives in a folder with its `README.md` and an optional `assets/` folder. No CMS, no database.

```bash
~/axison.dev/posts/
└── 2026-04-22_welcome/
    ├── README.md
    └── assets/
```

Try the **view modes** in the top right — there's GitHub, Codex, Claude, Antigravity, VS Code and Terminal. The Terminal one is the most fun. ✨
